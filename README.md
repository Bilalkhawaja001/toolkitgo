# Free Tools Website

40+ Free Online Tools for Work, Business & Development

## Features
- **40+ Tools** across 6 categories
- **100% Client-side** - No data leaves your device
- **Instant Results** - No server uploads, no waiting
- **Free Forever** - No hidden fees or subscriptions

## Categories
1. **Text Tools** (8 tools) - Word Counter, Character Counter, Case Converter, etc.
2. **Business Tools** (7 tools) - Invoice Generator, Receipt Generator, etc.
3. **Calculator Tools** (7 tools) - Financial, health, date calculators
4. **Developer Tools** (8 tools) - JSON Formatter, Base64 Encoder, etc.
5. **QR & Barcode Tools** (5 tools) - QR Code Generator, Barcode Generator
6. **Social Media Tools** (5 tools) - Caption Generator, Hashtag Generator

## Run Commands

### Install dependencies
```bash
npm install
```

### Start development server
```bash
npm start
```

### Build for production
```bash
npm run build
```

### Run tests
```bash
npm test
```
